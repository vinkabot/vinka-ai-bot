# AI Automation Lab

## Project 1: GitHub API Data Fetcher

Automated system that:
- Fetches data from GitHub API
- Saves responses to JSON
- Runs on schedule using Windows Task Scheduler

Tech stack:
- Python
- REST API
- Automation
- Scheduling

Use cases:
- Data monitoring
- Backend automation jobs
- Reporting systems
---

## How to run

1. Install dependencies:
pip install requests

2. Run script:
python github_data_saver.py

3. Schedule automation using Windows Task Scheduler
